**How to use this document**

This file contains all editable text from the Sixense website, organised by page and section. To make changes:

1. Edit the text directly in this document
2. Return the file to your developer
3. Changes will be merged back into the site

**Rules:**
- Only edit text between lines or under field labels — do not change the headings that describe the structure (e.g. "SECTION LABEL", "H1", "BODY")
- Do not delete section or field labels — they help the developer find the right place
- If you want to delete a whole block of text, write `[DELETE]` in its place
- Notes in `[square brackets]` are instructions — do not include them in your edits
